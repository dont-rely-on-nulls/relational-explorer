#+TITLE: Sakura REPL

A terminal REPL for the [[https://github.com/dont-rely-on-nulls/sakura][Sakura Relational Engine]].

* Features

- Interactive query execution
- Scrollable result display
- Mouse and keyboard navigation
- Auto-follow latest results

* Usage

Press ~e~ to enter editing mode, type your query, and press ~Enter~ to execute.

Use arrow keys or scroll wheel to navigate through query history.
